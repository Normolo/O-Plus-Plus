## Installation
1. Download and execute `installer.py`
2. Wait a bit while the program boots
3. Enter `install`
4. You now have O++!

```diff
- Before installation
Welcome to the O++ Installer
You will now be entering a Command Line Interface in order to install O++
OPP//: : install
```

```diff
